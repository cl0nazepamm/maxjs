# maxjs

Dual architecture three.js integration for 3dsmax. Idea is to turn 3dsmax into full blown high performance three.js editor. 


# Features

WebGPU backend
True realtime sync, extremely high performance.
GPU instancing
Three.js lights and materials available and ready to use
Guassian splat loader
Physical material, openpbr material, glTF material, vraymtl, materialX material, usdpreviewsurface supported
Cool postfx built in
Your 3dsmax scene is separate container so you can build three.js without breaking stuff. Allows you to play games directly in the viewport with realtime updates!!
WebXR virtual reality mode.
