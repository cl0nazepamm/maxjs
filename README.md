# max.js

Work with three.js directly inside 3dsmax with a natural interface. Dual architecture design allows you to add custom fx or anything really. There is no glTF gymnastic it directly writes from 3dsmax data.

It's registered as a renderer. You can use activeshade or external max.js panel. 
Comes with a nice postfx stack and it runs pretty well.

# Features

- Layer Manager
- Snapshots
- Natural Integration. It's genuinely a better integration than some renderers.
- Wide material support including Vraymtl
